# HealthStore - Wellness E-commerce Template

HealthStore is a modern, responsive, and elegant e-commerce landing page template designed for wellness and health-related brands. It features a clean aesthetic with smooth animations and a focus on user experience.

## Features

- **Modern Tech Stack**: Built with Tailwind CSS for styling and Alpine.js for lightweight interactivity.
- **Dynamic Animations**: 
  - Smooth "Fly to Cart" animations powered by Anime.js.
  - Interactive typing effect in the hero section.
  - Intersection Observer based scroll-reveal animations for sections.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices.
- **Interactive Cart**: Functional shopping cart UI (logic handled via Alpine.js) with quantity management and total calculation.
- **Micro-interactions**: 
  - Parallax background effect.
  - Sticky navigation with backdrop blur.
  - Hover effects on product cards.
  - "Back to Top" button with visibility toggle.
- **Design Elements**: 
  - Uses Phosphor Icons for a consistent look.
  - Inter font for high readability.
  - Soft gradient mesh background.

## Use Cases

This template is ideal for:
- Holistic health and wellness shops.
- Organic supplement brands.
- Herbal tea or natural remedy stores.
- Fitness and lifestyle landing pages.
- Health-conscious e-commerce startups.
