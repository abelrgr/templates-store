# Global Language Academy - Multi-Language School

A clean, light, and versatile template for language academies, ESL centers, and linguistics schools.

## Features
- **Minimal & Elegant Design:** Sky blue palette with spacious layouts and clean typography.
- **Interactive Language Courses:** Filterable course catalog by level (A1-C2) and language using Alpine.js.
- **Cultural Immersive Section:** Highlights on study abroad programs and international exchanges.
- **Teacher Spotlight:** Profiles of native-speaking instructors with expandable bios.
- **Testimonial Carousel:** Global student reviews handled with smooth Alpine.js transitions.
- **Simple & Fast Navigation:** Mobile-ready menu with light/dark theme toggle.
- **Validated Contact System:** Simple inquiry form with instant feedback via Alpine.js.

## Use Cases
- Language Hubs
- ESL & EFL Centers
- Study Abroad Agencies
- Translation & Linguistics Colleges
- Personal Tutoring Services

## Technologies Used
- **Tailwind CSS:** Responsive grid and minimal utility styles.
- **Alpine.js:** State management for the course catalog and interactive teacher profiles.
- **Tabler Icons:** Clean, modern linguistics and message icons.
- **Google Fonts:** Montserrat (Sans-serif) and Lora (Serif).
