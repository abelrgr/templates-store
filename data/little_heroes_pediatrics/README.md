# Little Heroes Pediatrics - Kid-Centric Healthcare

A warm, playful, and reassuring template designed for pediatric clinics, neonatal specialists, and child therapists.

## Features
- **Family-Friendly Aesthetic:** Soft rounded corners, primary color accents (Rose/Amber/Sky), and playful illustrations.
- **Vaccination & Milestone Trackers:** Interactive infographic tools powered by Alpine.js for parent education.
- **Secure Appointment Portal:** Easy scheduling for check-ups and urgent sick visits.
- **Televisit Integration:** Quick links and status indicators for remote consultations.
- **Kid's Zone Section:** Distraction resources, health games, and clinic preparation tips.
- **Doctor Meet-and-Greet:** Visual bios of pediatricians to build trust before the visit.
- **Responsive "Sick Visit" Button:** A high-contrast mobile header button for urgent medical needs.

## Use Cases
- General Pediatric Practices
- Pediatric Specialists (Dentists, Surgeons, etc.)
- Child Psychology Centers
- Neonatal Care Units

## Technologies Used
- **Tailwind CSS:** Custom configuration with "super-rounded" utilities and soft shadows.
- **Alpine.js:** State management for the vaccination schedule interactive and milestone modals.
- **Tabler Icons:** Hearts, babies, and medical icons for children.
- **Google Fonts:** Fredoka (Rounded, friendly) and Baloo 2.
