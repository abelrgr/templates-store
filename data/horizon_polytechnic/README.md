# Horizon Polytechnic - Technical & Vocational Institute

A bold, modern, and high-tech template for vocational colleges, engineering schools, and certification centers.

## Features
- **Technical Design Aesthetic:** Industrial slate colors with electric purple accents.
- **Certification Paths:** Grid-based layout for various technical certifications with interactive tooltips.
- **Industry Partners:** Section highlighting company collaborations and internships.
- **Workshop Highlights:** Responsive carousel for campus lab facilities and heavy machinery training.
- **Skill-Based Navigation:** Clear categories for different trade and tech sectors.
- **Dark Mode Optimization:** Neon-accented dark mode perfect for tech-focused audiences.
- **Validated Contact System:** Multi-step inquiry form for different department orientations.

## Use Cases
- Technical Colleges
- Vocational Training Centers
- Coding Bootcamps
- Engineering Institutes
- Trade Schools

## Technologies Used
- **Tailwind CSS:** Dynamic grid and industrial styling.
- **Alpine.js:** Advanced state for certification accordions and navigation logic.
- **Tabler Icons:** Engineering and technology-centric icons.
- **Unsplash:** High-quality industrial and tech imagery.
