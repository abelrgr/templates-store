# Bright Minds Elementary - School Template

A vibrant, friendly, and comprehensive school template designed for elementary education. Focused on early learning and parental engagement.

## Features
- **Vibrant & Accessible Design:** Playful layout with child-friendly color palettes and accessible typography.
- **Interactive Academic Programs:** Expandable details for each grade level and subject using Alpine.js.
- **Responsive Navigation:** Mobile-ready navigation menu with sticky positioning.
- **Dark Mode Support:** Smooth light/dark mode transitions built into the Tailwind structure.
- **Dynamic FAQ:** Accordion-style FAQ section for parent inquiries.
- **Validated Contact Form:** Interactive form with Alpine.js-powered success/error notifications.
- **Integrated Iconography:** Harmony with Tabler Icons across all sections.

## Use Cases
- Elementary Schools
- Early Learning Centers
- After-school Programs
- Educational Academies
- Primary Education Institutions

## Technologies Used
- **Tailwind CSS:** Modern utility-first CSS framework for styles.
- **Alpine.js:** Lightweight JavaScript interactivity for state management and UI components.
- **Tabler Icons:** Standardized icon set via CDN.
- **HTML5:** Semantic architecture for better SEO and accessibility.
