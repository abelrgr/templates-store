# Wonderland Toy Factory Template

A magical e-commerce experience for toys and childhood wonders.

## Features
- **Explosive Color Palette**: Rose red, amber yellow, and emerald green.
- **Bouncy UI Components**: Playful hover effects using Alpine.js.
- **Gift Finder**: Step-by-step interactive questionnaire for finding the perfect toy.
- **Safety Badge Hub**: Clearly visible age-appropriateness and safety ratings.

## Tech Stack
- Tailwind CSS
- Alpine.js
- Tabler Icons
- CSS Keyframe animations (bouncing/floating)
