# ElectroNexus Tech Hub Template

Cutting-edge electronics store showcase for modern tech enthusiasts.

## Features
- **Cyber-Industrial Design**: Deep blues and high-contrast yellow accents.
- **Spec-Heavy Layout**: Built to showcase technical details (CPU, RAM, etc.) clearly.
- **Interactive Comparison**: Alpine.js powered product comparisons.
- **Gamified Elements**: Pulse animations and tech-inspired grid backgrounds.

## Tech Stack
- Tailwind CSS
- Alpine.js
- Tabler Icons
- Tech-style glassmorphism filters
