# Atlas University - Higher Education Research Template

A global, elite, and versatile template for universities, colleges, and research institutions.

## Features
- **Elite Brand Identity:** Crimson and gold color scheme with a prestigious academic feel.
- **Academic Catalog:** Multi-layered project/research highlights and degree catalogs using Alpine.js.
- **Global Research:** Section for international project collaboration and research publications.
- **Campus Life:** Showcase residential life, student unions, and campus amenities.
- **Advanced Navigation:** Sophisticated mega-menu style navigation (mobile optimized).
- **Dark Mode Excellence:** High-contrast dark mode focusing on readability and professionalism.
- **Responsive Faculty Directory:** Filterable or expandable faculty details using Alpine.js.

## Use Cases
- State & Private Universities
- Global Research Centers
- Graduate Schools
- Liberal Arts Colleges
- Medical & Law Institutes

## Technologies Used
- **Tailwind CSS:** Modern grid systems and responsive utility classes.
- **Alpine.js:** Modal management, state handling for research catalogs, and navigation.
- **Tabler Icons:** Comprehensive icon set.
- **Google Fonts:** Playfair Display (Serif) and Inter (Sans).
