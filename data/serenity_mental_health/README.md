# Serenity Mental Health & Wellness Template

Professional, compassionate website template for mental health professionals and clinics.

## Features
- **Calming Design**: Sage green and lavender palette to reduce anxiety.
- **Provider Directory**: Easy-to-navigate therapist profiles.
- **Patient Resource**: Built-in mood tracking and meditation links.
- **Safety First**: Clearly visible crisis contact information and quick-exit button.

## Tech Stack
- Tailwind CSS
- Alpine.js
- Tabler Icons
- Dark Mode support (Deep Slate / Night Moss)
