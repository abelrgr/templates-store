# Pure Smile Dentistry - Modern Oral Care

A bright, inviting template specifically designed for dental practices, orthodontists, and cosmetic surgeons.

## Features
- **Inviting & Clean UI:** Soft teal and cyan color palette to reduce patient anxiety.
- **Interactive Service Blocks:** Detailed breakdowns of treatments (Implants, Braces, Whitening) with hover effects.
- **Before/After Showcase:** Custom slider/comparison component for cosmetic results.
- **Appointment Request Flow:** Multi-step consultation form using Alpine.js.
- **FAQ Section:** Dynamic accordion for common patient questions about insurance and procedures.
- **Emergency Dental Contact:** Floating contact action for urgent care needs.
- **Testimonial Grid:** Focus on smile transformations and patient comfort stories.

## Use Cases
- Family Dental Practices
- Cosmetic Dentistry Studios
- Orthodontic Clinics
- Periodontal Specialists

## Technologies Used
- **Tailwind CSS:** Responsive layout with custom "cyan-soft" gradients.
- **Alpine.js:** Handles the multi-step form, FAQ accordions, and mobile menu.
- **Tabler Icons:** Specialized dental and medical icon set.
- **Google Fonts:** Quicksand (Rounded, friendly Sans-serif) and Bitter (Reliable Serif).
