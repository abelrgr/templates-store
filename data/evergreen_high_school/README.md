# Evergreen High School - Secondary Education Template

A modern, professional, and comprehensive template tailored for high schools and preparatory institutes.

## Features
- **Professional Academic Design:** Clean grid-based layouts and deep forest green accents.
- **Academic Departments:** Detailed section for departments with interactive faculty highlights.
- **Athletics & Clubs:** Showcase sports achievements and extracurricular student life.
- **Modern Navigation:** Multi-level mobile-friendly menu with dark mode support.
- **Interactive FAQ:** Common parent/student questions handled via Alpine.js.
- **Career Toolbox:** Resource downloads for students applying to universities.

## Use Cases
- High Schools
- Secondary Schools
- Preparatory Academies
- International Secondary Institutes

## Technologies Used
- **Tailwind CSS:** Styles and responsive utility classes.
- **Alpine.js:** Modal handling, tab switching, and general light state management.
- **Tabler Icons:** Comprehensive icon set.
- **Unsplash:** High-quality academic imagery.
