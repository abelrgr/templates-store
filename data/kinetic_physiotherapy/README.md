# Kinetic Physical Therapy & Rehab Template

Dynamic and modern template for physiotherapy clinics and sports rehabilitation centers.

## Features
- **Action-Oriented Design**: High-contrast blue and orange palette for energy.
- **Recovery Roadmap**: Visual timeline for patient rehabilitation progress.
- **Education Hub**: Specialized sections for exercise videos and injury prevention.
- **Performance Focused**: Includes tools for athletes to track return-to-play metrics.

## Tech Stack
- Tailwind CSS
- Alpine.js
- Tabler Icons
- Performance-focused animations (Framer-motion inspired via Tailwind)
