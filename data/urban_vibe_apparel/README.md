# UrbanVibe Streetwear Template

The ultimate digital storefront for high-heat streetwear and youth fashion.

## Features
- **Brutalism-Inspired Design**: Stark black/white contrast with neon accents.
- **Dynamic Drop System**: Built-in countdowns for limited-edition releases.
- **Micro-Interactions**: Smooth hover effects and liquid-style transitions.
- **Social Integration**: Native-feeling Instagram and TikTok style layouts.

## Tech Stack
- Tailwind CSS
- Alpine.js
- Tabler Icons
- Inter & Syne font pairing for max impact.
