# Vision Precision Optometry Template

Sleek, glassmorphism-inspired template for modern optometry practices and premium eyewear boutiques.

## Features
- **Crystal Clear UI**: Heavy use of glassmorphism and high-contrast typography.
- **Vision Simulator**: Interactive Alpine.js tool to demonstrate different eye conditions.
- **Eyewear Catalog**: Elegant grid layout for glasses and sunglasses.
- **Tech-Forward**: Highlights advanced diagnostic tools (Retinal scanning, etc.).

## Tech Stack
- Tailwind CSS
- Alpine.js
- Tabler Icons
- Glassmorphism effects (backdrop-blur)
