# Alpaca Heritage Luxury Template

Elevated e-commerce for the world's finest sustainable wool collections.

## Features
- **Sophisticated Earthy Palette**: Taupe, beige, and deep chocolate brown.
- **Micro-Copy Focus**: Highly polished typography for a quiet luxury feel.
- **Heritage Sections**: Dedicated layouts for fiber origin and weaving techniques.
- **Glassmorphism Textures**: Soft UI elements that mimic the transparency of fine knits.

## Tech Stack
- Tailwind CSS
- Alpine.js
- Tabler Icons
- Editorial-style typography (Playfair Display & Montserrat)
